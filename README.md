# Recommendation-systems
## Dataset

This project uses publicly available datasets.

### Movie Recommendation Dataset
- TMDB 5000 Movie Dataset:
  https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

### Collaborative Filtering Dataset
- MovieLens Latest Small:
  https://grouplens.org/datasets/movielens/latest/

### Fashion Recommendation Dataset
- Myntra Fashion Clothing Dataset:
  https://www.kaggle.com/datasets/shivamb/fashion-clothing-products-catalog

## Dataset Setup

Download the datasets manually and place them in the `data/` folder.

Project Structure:

```text
project/
│
├── data/
│   ├── tmdb_5000_movies.csv
│   ├── tmdb_5000_credits.csv
│   ├── movies.csv
│   ├── ratings.csv
│   └── styles.csv
│
├── Content_based_filtering.py
├── Collaborative_filtering.py
├── Hybrid_filtering.py
└── requirements.txt
```
